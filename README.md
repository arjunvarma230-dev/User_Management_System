# User_Management_System
A simple console-based User Management System built with Python and MySQL. This project demonstrates modular design, database CRUD operations, user authentication, and custom exception handling.

# User Management System

**Description:**  
This is a **console-based User Management System** built in Python. It allows users to **register, login, view their details, change password or address, and delete their account**.

**Tools Used:**  
- **Python 3** – for application logic and menu interface  
- **MySQL** – to store user data  
- **mysql-connector-python** – to connect Python with MySQL  
- **MySQL Workbench (optional)** – to test and manage the database

It’s a modular project demonstrating **backend logic, database CRUD operations, and exception handling**.
